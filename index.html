Images (JPEG / PNG / WebP)

The file is redrawn onto a blank canvas and re-exported. Only pixel data survives, so this reliably strips EXIF (GPS, camera model, timestamps, software tag), ICC color profiles, XMP, IPTC, and embedded thumbnails.

Trade-off: the image is recompressed, so there's a small, usually invisible quality loss.

PDF

Clears the document info dictionary (title, author, subject, producer, creator, dates), removes the XMP metadata stream, and strips embedded files, attachments, and open-action JavaScript where present.

Limitation: form field data, digital signatures, and content baked into page objects (like a scanned watermark) are not touched. For those, a desktop tool such as ExifTool or MAT2 goes further.

Not yet supported

TIFF, HEIC, GIF, Office files (.docx/.xlsx/.pptx), audio, and video. These are flagged as unsupported rather than silently passed through.

Checking what's there first

MetaLyzer (linked above the file list) is a separate viewer for inspecting EXIF and GPS data before you decide what to clean. It's a different site, so treat it as a separate trust boundary from this tool.
